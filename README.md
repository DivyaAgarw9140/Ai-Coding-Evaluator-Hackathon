## üåü Overview
Elite AI Evaluator is a professional-grade automated code assessment tool. It goes beyond simple test cases to identify **Engineering Signals** like logic depth, algorithmic complexity, and code maintainability, providing recruiters with instant, fair, and deep insights.

## ‚ú® Key Features
- **Persona-Based Evaluation:** Choose between a 'Google Senior Engineer', 'Startup CTO', or 'Clean Code Expert' to get different perspectives on the code.
- **Visual Performance Dashboard:** Instant scoring across 5 dimensions: Correctness, Efficiency, Readability, Modularity, and Edge Cases.
- **Skill Mapping:** A visual bar chart to identify a candidate's technical profile at a glance.
- **Automated Refactoring:** Generates a "Perfect Industry Version" of the submitted code to help students learn.
- **Plagiarism Detection:** Analyzes code for common tutorial patterns and boilerplate solutions.
